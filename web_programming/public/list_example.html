토마토 달걀 볶음

토마토 달걀 볶음, 중국어로 시홍스차오지단 은 중국의 요리이다.
본요리로 나오는 것 중 하나로, 간단하고 빠르게 요리할 수 있기 때문에 학생들 사이에서 인기가 높은 요리이다.

재료

  • 계란
  • 방울토마토
  • 쪽파
  • 소금

방법

  1. 우선 달걀은 사발에 깨넣어 섞고, 방울토마토와 쪽파는 조각조각 자른다.
  2. 그런 뒤 기름을 두른 프라이팬에 토마토를 약 1분간 볶고 소금을 뿌려 간을 맞춘 다음 마지막에 계란을 넣는다.
  3. 소금과 후춧가루를 간을 맞추고, 완전히 다 익을 때까지 볶는다.
